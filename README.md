Introducing Vault: A secure real-time room-based communication platform that allows users to communicate seamlessly with other through messages on a real time basis.

<b>Objectives:</b>
<ul>
  <li>Users can join an existing room by entering a room ID.</li>
  <li>Users can create a new room with a unique room ID.</li>
  <li>Messages within a room are stored in MongoDB database for persistence.</li>
  <li>Users can see the previously saved messages when they join a room.</li>
  <li>Enable real-time messaging updates for all users in the room with secure and reliable protocols(Web Socket Protocol).</li>
  <li>React is used for creating a responsive and dynamic UI and Spring Boot is used for backend part of the application.</li>
</ul>
